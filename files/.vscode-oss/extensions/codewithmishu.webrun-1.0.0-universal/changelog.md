# Change Log

All notable changes to the "WebRun" extension will be documented in this file.

## [1.0.0] - 2026-02-04

### Added
- 🎉 Initial release of WebRun
- 🔍 Auto-detection of project types from package.json, config files, and folder structure
- ▶️ One-click status bar button to start dev servers
- ⏹️ Stop and restart functionality
- 📦 Support for 15+ project types:
  - Frontend: React (Vite/CRA), Vue (Vite/CLI), Next.js, Nuxt.js, Angular, Svelte, Astro, Remix
  - Backend (Node.js): Express.js, Fastify, NestJS
  - Backend (Python): Flask, Django, FastAPI
  - Static: Plain HTML/CSS/JS
- 🔀 Fullstack project support (frontend + backend in separate terminals)
- ⌨️ Keyboard shortcuts for quick access
- 🎛️ Configuration options for custom commands
- 📋 Command palette integration
- 🖱️ Manual project type selection
